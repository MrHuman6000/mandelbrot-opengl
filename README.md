# Mandelbrot OpenGL Renderer

Real-time Mandelbrot fractal renderer written in **C++** using **OpenGL**.

The fractal is generated entirely on the **GPU using GLSL shaders**.

---

## Preview

Real-time animated Mandelbrot fractal.

---

## Features

- Real-time Mandelbrot fractal rendering
- GPU accelerated using GLSL
- Animated zoom
- Color palette using sin / cos functions
- FPS counter in window title
- Fullscreen quad rendering

---

## Technologies

- C++
- OpenGL 3.3
- GLFW
- GLAD
- GLSL
